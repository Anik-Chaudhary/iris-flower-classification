# Iris Flower Classification

Machine Learning classification project completed as part of the **QSkill AI/ML June–July Internship Task**.

---

## Objective

Predict the species of an Iris flower using four flower measurements:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

The three classes are:

- Setosa
- Versicolor
- Virginica

---

## Dataset

This project uses the Iris Dataset available directly from Scikit-learn.

Number of Samples: **150**

Number of Features: **4**

Classes:

- Setosa
- Versicolor
- Virginica
